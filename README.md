# Projeto PITES II: Gerenciador de Tarefas

**Objetivo:** Criar uma aplicação web para gerenciar tarefas pessoais (To-Do List), permitindo ao usuário cadastrar, editar, excluir e marcar tarefas como concluídas.

**Público-Alvo:** Usuários que precisam de uma ferramenta simples para organizar seu dia-a-dia.

**Plataforma:** Aplicação Web (MVC)
